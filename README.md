☕ CoffeeShop
Practical task at the end of the module: "The Complete JavaScript + React Course".

A modern e-commerce coffee shop web application built with React + Vite.
The app allows users to browse coffee products, add items to a cart, and complete a checkout flow.

👉 Live Demo: https://itssonyamisyura.github.io/CoffeeShop/

👉 Repository: https://github.com/itssonyamisyura/CoffeeShop

🚀 About

This project is a fully functional frontend application that simulates an online coffee store.

It demonstrates:

Building scalable React applications
Managing global state (cart logic)
Working with routing and dynamic pages
Persisting data using localStorage

✨ Features
🏠 Home page with modern UI
☕ Coffee & goods catalog
🔍 Product details page
🛒 Shopping cart with:
Add / remove items
Increase / decrease quantity
Total price calculation
Undo clear cart
💾 Cart persistence via localStorage
🧾 Checkout page
🚫 Custom 404 page
📱 Fully responsive design
🎬 Scroll animations (AOS)

🛠️ Technologies Used
Frontend
React 19
React Router DOM (routing)
Vite (build tool)

UI & Styling
CSS / SCSS
